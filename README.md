# Cardash 🚗

El marketplace de autos de Venezuela. Compra, vende y alquila con vendedores reales y verificados.

Cardash es la versión para autos del molde de [Propiedash](https://propiedash.com): mismo lenguaje visual (verde lima sobre negro), adaptado al mundo automotor.

## Páginas

| Página | Descripción |
|--------|-------------|
| `index.html` | Home: hero con búsqueda, tipos de vehículo, autos destacados, directorio, guías |
| `buscar.html` | Resultados de búsqueda con filtros funcionales (operación, tipo, marca, precio, año, transmisión, combustible, km) |
| `auto.html` | Ficha individual del auto: galería, ficha técnica, equipamiento, contacto y autos similares |
| `publicar.html` | Formulario para publicar un auto: wizard de 5 pasos con vista previa en vivo |
| `directorio.html` | Directorio de profesionales: concesionarios, vendedores, talleres, peritos, aseguradoras y financiamiento |

## Cómo verlo

Es HTML, CSS y JavaScript sin dependencias. Desde la carpeta del proyecto:

```bash
python3 -m http.server 8777
```

Y abre `http://localhost:8777` en tu navegador.
